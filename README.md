# 🧩 CodeMaster: Учимся программировать

Полноэкранное образовательное приложение на `pygame` для обучения детей основам алгоритмического мышления. Ребёнок пишет простые команды, а красный квадратик выполняет их в игровом поле с пошаговой визуализацией и подсветкой текущей строки кода.

---

## ✨ Возможности
- 📝 **Простой редактор** с поддержкой команд `up()`, `down()`, `left()`, `right()`
- 🎮 **Игровое поле 10×10** с пошаговым выполнением и анимацией перемещения
- 🔍 **Валидация кода**: подсветка строки с ошибкой и понятное сообщение
- 💾 **Автосохранение**: код загружается при старте и сохраняется при запуске/очистке/выходе
- 🧹 **Кнопка `CLEAR`** для быстрого сброса редактора
- 🧪 **Покрыт тестами** (парсер + движок выполнения)
- 🌐 Поддержка кириллицы в сообщениях и интерфейсе

---

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
uv sync
```

### 2. Запуск приложения
```bash
uv run main.py
```

### 3. Как играть
1. Пишите команды в правой панели (по одной на строку)
2. Нажмите `▶ RUN` для запуска выполнения
3. Следите за перемещением квадратика и подсветкой строки
4. `✕ CLEAR` очищает редактор, `ESC` закрывает приложение

## 📂 Структура проекта
```
codemaster/
├── main.py          # Точка входа, игровой цикл
├── code_parser.py   # Парсинг и валидация команд
├── game.py          # Логика сетки и перемещения
├── engine.py        # Пошаговый движок выполнения
├── ui.py            # Рендеринг, ввод, кнопки, курсор
├── storage.py       # Сохранение/загрузка кода (UTF-8)
└── tests/           # Модульные тесты (unittest)
```

---

## 🧪 Тестирование
```bash
python -m unittest discover tests/ -v
```
Проверяют корректность парсинга команд, обработку ошибок, пошаговое выполнение и ограничение границ поля.

## ⚙️ Технические детали
- **Python**: `3.9+`
- **GUI**: `pygame 2.6.1+`
- **Режим**: Полноэкранный (`pygame.FULLSCREEN`)
- **Шрифты**: Автоматический подбор системного моноширинного шрифта с поддержкой кириллицы (`Menlo` / `Courier New` / `DejaVu Sans Mono`)
- **Хранение кода**: `saved_code.txt` в корне проекта
